# @vuedapt/create-node

A CLI tool to quickly initialize a new Node.js application.

## Installation

This package is designed to be used with `npm init`:

```bash
npm init @vuedapt/node
```

## Usage

Run the following command to create a new Node.js project:

```bash
npm init @vuedapt/node [project-name]
```

The tool will prompt you for:
- Project name
- Description
- Version
- Author
- License
- Package manager (npm, yarn, or pnpm)
- Whether to install dependencies

## What it creates

The tool will create a new Node.js project with:

- `package.json` - Project configuration
- `index.js` - Main entry point
- `README.md` - Project documentation
- `.gitignore` - Git ignore file

## Features

- Interactive prompts for project configuration
- Support for npm, yarn, and pnpm
- Automatic dependency installation
- Modern project structure
- Git-ready setup
